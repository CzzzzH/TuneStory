请先准备好如下材料以制作谱面：
1. 谱面音乐文件（仅限mp3, wma, wav三种格式）
2. 谱面小图（显示在主菜单上的，最好为正方形的分辨率，格式仅限png）
3. 谱面背景图片（最好为16 : 9的分辨率，格式仅限png）

做谱步骤如下：
①在根目录下的Songs文件夹创建一个新的文件夹，名称命名为谱面名称
②将1,2,3文件全部放入新创建的文件夹中，将1和2命名为谱面名称，将3命名为谱面名称_B
③创建一个新文件，命名为谱面名称（后缀为.tsm），使用记事本对该文件按照一定格式进行编辑。格式如下（括号内为标注，不含在内）：

====文件开始====
<歌曲作者>
<谱面作者>
<歌曲BPM>（Beat Per Minute，即每分钟节拍数）
<谱面所含难度>（一个四位01串，表明含有哪些难度，按Easy, Normal, Hard, Extreme排序，例如你的谱面含有Normal和Hard难度，即填写0110）

（如有K个不同难度，接下来按与01串相同的顺序依次填写K个难度的谱面顺序）

<谱面Note个数>  <难度等级> （Note即为需要打击的音符，难度等级是难度的细分，为1~12的数字，除此之外没有严格的范围要求）

（如有N个Note，接下来则有N行，下面以第i行为例（1 <= i <= N）
<第i个音符的X坐标> <第i个音符的Y坐标> <第i个音符的颜色样式> <第i个音符的打击时间> <第i个音符的长按标识>
（需满足 0 <= X <= 1920， 0 <= Y <= 1080）
（颜色为Red, Blue, Green之一，样式为1, 2, 3之一，颜色和样式之间没有空格）
（打击时间单位为秒，等于歌曲已经进行的时间，保留三位小数）
（由于现在游戏还没有长按机制，故长按标识填写数字0即可）
（***请不要在间隔过短的时间里放两个及以上的绿色音符，因为绿色音符只有一个键响应，打不了那么快***）
====文件结束====

文件夹里的最后一个tsd文件存储了玩家的本地成绩，由游戏系统自己创建，无需自己创建（当然想删就可以删），当谱面内容改动时会自动重建。

***请严格按照上述格式填写谱面文件，否则会导致游戏崩溃***
***游戏目录下的Beatmap Maker可以将Osu! Standard中的谱面转化为TS谱面（感谢Spitlight提供），运行该程序后只需将Osu的谱面文件中的HitObject下所有内容复制至程序中按q即可（其中Osu的普通Note音会被自动转化为TS中的红色Note，Whistle音会被自动转化为TS中的绿色Note，Finishi音会被转化为TS的蓝色Note，但Note样式均默认为1）
***如有不清楚的地方可以打开已有的谱面参考，还是不清楚就联系作者询问（点击游戏内的Help按钮可查看作者的联系方式）***
